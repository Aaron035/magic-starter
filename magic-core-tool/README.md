## 简介

`magic-core-tool` 主要提供常用的 API 返回参数封装，分页条件封装，分页结果封装，通用异常封装，常用工具类封装，同时集成 `Google Guava`、`Hutool`等工具包。

## 使用

```xml
<dependency>
  <groupId>com.xkcoding</groupId>
  <artifactId>magic-core-tool</artifactId>
  <version>${magic-starter.version}</version>
</dependency>
```
